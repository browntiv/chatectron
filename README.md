## Using
Install dependencies
```shell
npm install
```
Run the server
```shell
node server
```
Run the app in another shell
```shell
npm start
```
To build the app
```shell
npm win32,win64,linux32,linux64,osx
```
Choose one of them or build with `npm all` for every system
